# TaskBoard (KatBan)

TaskBoard (KatBan) es una aplicación web inspirada en la metodología Kanban que permite organizar proyectos mediante tableros, columnas y tareas. Permite crear y administrar proyectos, gestionar tareas, moverlas entre etapas mediante drag & drop y realizar búsquedas de forma rápida e intuitiva.

El proyecto evolucionó desde una versión desarrollada con **LocalStorage** hacia una arquitectura **Full Stack**, incorporando un **backend con Node.js, Express.js y MongoDB**. Gracias a esta implementación, la información de usuarios, proyectos y tareas se almacena en una base de datos, ofreciendo una gestión más segura, escalable y persistente. :contentReference[oaicite:0]{index=0}

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Bootstrap Icons
- Node.js
- Express.js
- MongoDB

## Funcionalidades

- Registro e inicio de sesión de usuarios.
- Creación, edición y eliminación de proyectos.
- Gestión de columnas y tareas.
- Arrastrar y soltar tareas entre columnas.
- Buscador de proyectos y tareas.
- Persistencia de datos mediante MongoDB.
- API REST para la comunicación entre el frontend y el backend.
